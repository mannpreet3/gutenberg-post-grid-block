# gutenberg-post-grid-block

A flexible Gutenberg block that displays posts from any post type in a responsive grid.

## Features
- Select any post type (Posts, Pages, Custom Post Types)
- Responsive grid layout (1-6 columns)
- Customizable display options
- Category filtering for posts
- Featured image support
- Author and date display

## Installation
1. Download the plugin
2. Upload to `/wp-content/plugins/`
3. Activate in WordPress admin

## Usage
1. Add "Post Grid" block in Gutenberg editor
2. Select post type from sidebar
3. Configure query and display settings
4. Save and view
